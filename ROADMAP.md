# Roadmap publico de ONE Community Edition

## 0.1 - Primer paquete comunitario

- CLI basica.
- ONE Studio basico.
- Plantillas iniciales.
- Generador web local.
- Documentacion inicial.

## 0.2 - Instaladores simples

- Script para Windows.
- Script para macOS.
- Script para Linux.
- Paquete descargable por sistema.

## 0.3 - Studio mas facil

- Editor por formularios.
- Vista previa antes de construir.
- Boton para exportar proyecto.
- Boton para importar proyecto.

## 0.4 - Mas plantillas

- Agenda.
- Inventario avanzado.
- Directorio.
- Formulario.
- Encuesta.

## 1.0 - Primera version estable

ONE 1.0 debe permitir que una persona no programadora pueda abrir ONE Studio, elegir plantilla, modificar campos, construir una app, guardar datos, exportar el proyecto y compartirlo.
